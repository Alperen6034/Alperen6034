## Selected Engineering Work

### Real-Time Robot Goalkeeper Control System
STM32-based motion-control and actuation system using STEP/DIR motor control,
UART communication, trajectory commands and mechanical transmission analysis.

### Attitude Determination System
STM32-based sensor acquisition, calibration, UART telemetry and quaternion-based
attitude estimation using inertial and magnetic sensors.

### 8-bit Absolute Optical Encoder
Absolute encoder development using a Gray-code optical disk, IR sensing,
signal conditioning and STM32-based position decoding.

### EtherCAT Servo Control
EtherCAT master development experience using SOEM, CiA-402 servo drives,
PDO/SDO communication and UDP-based telemetry.
